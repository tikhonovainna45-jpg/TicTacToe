# Отзыв по проекту «TicTacToe»

## Инна

**Что сделала:** игра «Крестики-нолики» для двух игроков на C++ — проверка 8 линий побед, определение ничьи, возможность переиграть.

**Сильные стороны:**
- Правильная игровая логика.
- Понятные имена переменных (`brd`, `cur`).
- Русскоязычный дружелюбный интерфейс.

**Что можно улучшить:**
- В нескольких местах (строки 9–19, 30–32) пропущен `endl` или `"\n"` — без этого программа не скомпилируется. Например: `cout << "Ходит игрок" << cur << endl;`.
- На строке 45 между условиями нужен `||`: `mvv < 1 || mvv > 9`.
- На строке 102 аналогично — добавь `||` между вариантами ответа.
- Блок вывода доски дублируется (строки 29–32 / 35–40) — вынеси в отдельную функцию `void drawBoard()`.
